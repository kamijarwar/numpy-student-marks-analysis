# Student Marks Analysis Using NumPy

This is a simple Python project that analyzes student marks using the NumPy library.

## Features

- Calculate average marks
- Find highest marks
- Find lowest marks
- Display Pass/Fail status of each student
- Count total passed students
- Count total failed students

## Technologies Used

- Python
- NumPy

## Code Example

```python
import numpy as np

marks = np.array([88, 99, 67, 87, 65, 85, 75, 23, 33, 43])
```

## Sample Output

```
Average marks: 66.5
Highest marks: 99
Lowest marks: 23

Pass students: 8
Fail students: 2
```

## How to Run

1. Install NumPy

```bash
pip install numpy
```

2. Run the Python file

```bash
python student_analysis.py
```
